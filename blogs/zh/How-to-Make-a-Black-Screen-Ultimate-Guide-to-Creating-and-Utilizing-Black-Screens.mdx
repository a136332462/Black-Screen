## 了解黑屏的创建方法

创建黑屏不仅仅是让屏幕变黑，更是为了满足特定需求选择合适工具。以下是几种高效方法：

### 快捷键速成法

有时候你只需要一个小技巧，就能快速进入黑屏状态。

#### Windows

- **F11**：浏览器或图片查看器进入全屏，适合展示黑屏。
- 打开黑色图片或网页后按 **F11**，营造沉浸式体验。
- 使用 **PowerPoint** 全屏播放黑色幻灯片，适合演示。

#### macOS

- **Control + Command + F**：支持应用（如 Safari、预览、Keynote）进入全屏。
- 显示纯黑图片或幻灯片效果最佳。
- **Keynote** 黑屏幻灯片 + 暂停 = 专业级黑屏。

> 注意：**Windows + P** 或 **Ctrl + Shift + B** 等快捷键不建议用于黑屏，它们可能会更改显示设置或意外重置驱动。
