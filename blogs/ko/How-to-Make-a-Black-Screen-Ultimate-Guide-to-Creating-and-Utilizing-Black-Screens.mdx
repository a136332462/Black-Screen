## 블랙 스크린 제작 방법 이해하기

블랙 스크린을 만드는 것은 단순히 어둠을 표시하는 것이 아니라 특정 요구사항에 맞는 올바른 도구를 갖는 것입니다. 가장 효과적인 방법들을 탐구해보겠습니다.

### 빠른 키보드 단축키

때로는 완전한 도구가 필요하지 않고, 어둡게 가는 빠른 트릭만 있으면 됩니다.

#### Windows

- **F11**: 브라우저나 이미지 뷰어의 전체화면 모드 — 블랙 스크린 표시에 이상적
- 블랙 이미지나 웹페이지를 열고 **F11**을 눌러 몰입 효과 생성
- 프레젠테이션을 위해 **PowerPoint**에서 블랙 슬라이드를 전체화면 모드로 재생

#### macOS

- **Control + Command + F**: Safari, Preview, Keynote과 같은 지원 앱의 전체화면 모드
- 최적의 결과를 위해 전체화면에서 순수한 블랙 이미지나 슬라이드 표시
- **Keynote** 블랙 슬라이드 + 일시정지 = 전문적인 블랙 스크린

> 참고: **Windows + P** 또는 **Ctrl + Shift + B**와 같은 단축키는 디스플레이 설정을 변경하거나 의도치 않게 드라이버를 재설정할 수 있으므로 블랙 스크린 목적으로는 권장하지 않습니다.
