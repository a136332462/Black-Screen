## 了解黑螢幕的創建方法

創建黑螢幕不僅僅是讓螢幕變黑，更是為了滿足特定需求選擇合適工具。以下是幾種高效方法：

### 快捷鍵速成法

有時候你只需要一個小技巧，就能快速進入黑螢幕狀態。

#### Windows

- **F11**：瀏覽器或圖片檢視器進入全螢幕，適合展示黑螢幕。
- 開啟黑色圖片或網頁後按 **F11**，營造沉浸式體驗。
- 使用 **PowerPoint** 全螢幕播放黑色投影片，適合演示。

#### macOS

- **Control + Command + F**：支援應用（如 Safari、預覽、Keynote）進入全螢幕。
- 顯示純黑圖片或投影片效果最佳。
- **Keynote** 黑螢幕投影片 + 暫停 = 專業級黑螢幕。

> 注意：**Windows + P** 或 **Ctrl + Shift + B** 等快捷鍵不建議用於黑螢幕，它們可能會更改顯示設定或意外重置驅動。
