## サイト概要

BlackScreen は、集中、OLED 画面保護、クリーンなプレゼンに対応した無料で広告なしのオンライン全画面ツールです。Next.js 14 とモダンな Web 技術で構築され、BlackScreen はすべてのデバイスで即座にアクセス可能で、インストール不要です。このプラットフォームは、プロフェッショナル、教育者、クリエイター、そして気が散らない画面体験を求めるすべての人にサービスを提供します。

**主要な価値提案：**
- **ワンクリック全画面** - スペースキーで即座にアクティブ化
- **8K 高解像度** - プレミアムディスプレイデバイスをサポート
- **広告ゼロ、トラッキングゼロ** - プライバシー重視の体験
- **オフライン動作** - PWA 対応で信頼性の高いアクセス
- **多言語サポート** - 英語、日本語、中国語、ドイツ語、韓国語、アラビア語など 9 言語をサポート

## ウェブサイト体験とパフォーマンス最適化

### Core Web Vitals 最適化

- **LCP（最大コンテンツペイント）：** < 2.5 秒
- **INP（インタラクションから次のペイントまで）：** < 200 ミリ秒
- **CLS（累積レイアウトシフト）：** < 0.1

### 技術最適化

- **コード分割：** ルートベースの自動コード分割
- **レイジーローディング：** オンデマンドで画像とコンポーネントを読み込み
- **画像最適化：** WebP フォーマット、レスポンシブ画像
- **キャッシュ戦略：** ブラウザキャッシュとサービスワーカーでオフラインアクセスをサポート
- **SSR/SSG：** サーバーサイドレンダリングと静的サイト生成で高速な初回読み込み

### モバイル最適化

- **レスポンシブデザイン：** すべてのデバイスサイズに完全対応
- **タッチフレンドリーインターフェース：** モバイルインタラクション向けに最適化
- **PWA サポート：** プログレッシブ Web アプリ機能でオフラインアクセスをサポート
- **高速モバイルパフォーマンス：** モバイル Core Web Vitals 向けに最適化

### セキュリティとプライバシー

- **HTTPS：** すべてのページでセキュア接続
- **トラッキングなし：** サードパーティのトラッキングスクリプトゼロ
- **ローカルストレージ：** データをブラウザのローカルに保存
