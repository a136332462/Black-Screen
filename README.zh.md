## 网站简介

BlackScreen 是一款免费、无广告的在线全屏工具，专为专注工作、OLED 屏幕保护与纯净演示而设计。基于 Next.js 14 和现代 Web 技术构建，BlackScreen 可在所有设备上即时访问，无需安装。平台服务于专业人士、教育工作者、创作者以及任何寻求无干扰屏幕体验的用户。

**核心价值主张：**
- **一键全屏** - 通过空格键快速激活
- **8K 高分辨率** - 支持高端显示设备
- **零广告、零跟踪** - 隐私优先体验
- **离线可用** - 支持 PWA，可靠访问
- **多语言支持** - 支持 9 种语言，包括中文、英文、日文、德文、韩文、阿拉伯文等

## 网站体验与性能优化

### Core Web Vitals 优化

- **LCP（最大内容绘制）：** < 2.5 秒
- **INP（交互到下次绘制）：** < 200 毫秒
- **CLS（累积布局偏移）：** < 0.1

### 技术优化

- **代码分割：** 基于路由的自动代码分割
- **懒加载：** 按需加载图片和组件
- **图片优化：** WebP 格式，响应式图片
- **缓存策略：** 浏览器缓存和服务工作器支持离线访问
- **SSR/SSG：** 服务端渲染和静态站点生成，快速初始加载

### 移动端优化

- **响应式设计：** 完全适配所有设备尺寸
- **触摸友好界面：** 针对移动交互优化
- **PWA 支持：** 渐进式 Web 应用功能，支持离线访问
- **快速移动性能：** 针对移动端 Core Web Vitals 优化

### 安全与隐私

- **HTTPS：** 所有页面安全连接
- **无跟踪：** 零第三方跟踪脚本
- **本地存储：** 数据存储在浏览器本地
